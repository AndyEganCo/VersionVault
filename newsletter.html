We've been working on a few updates to make VersionVault work better for you.

You're in control of "all quiet" emails

You can now customize when you receive "all quiet" emails—those updates letting you know there are no new versions for your tracked software.

Head to Settings to choose your preference:

  • Always send – Get peace of mind knowing everything is up to date (default)
  • Only when new software is available – Only get "all quiet" emails when we add new software to the platform
  • Never send – Only email me when there are actual updates

We're tracking more software

We're continuously expanding our software catalog and improving version detection. Recent improvements include:

  • Doubled content extraction limits to capture more version information
  • Enhanced scraping accuracy for complex release pages
  • Better support for different software sources
